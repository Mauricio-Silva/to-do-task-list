■ Install Nest.JS:
npm i -g @nestjs/cli

■ Create Nest.JS New Project:
nest new {project-name}

■ Run Nest.JS App:
npm run start:dev

■ Generate a New Controller Package:
nest g controller {name} --no-spec

■ Generate a New Service Package:
nest g service {name} --no-spec

■ Generate a New Entity:
nest g resource {name} --no-spec
